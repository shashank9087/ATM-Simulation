# ATM Simulation System (Python)

A console-based ATM Simulation System developed using Python. This project simulates the basic functionalities of a real ATM machine and demonstrates the use of Python concepts such as object-oriented programming, file handling, JSON, exception handling, decorators, and modular programming.

---

## Features

* User Registration
* Secure Login System
* Account Number & PIN Authentication
* Balance Inquiry
* Cash Deposit
* Cash Withdrawal
* Fund Transfer
* Change ATM PIN
* Transaction History
* Mini Statement
* Persistent Data Storage using JSON
* Error Handling & Input Validation
* Modular Code Structure

## How It Works

1. Create a new account.
2. An Account Number and ATM PIN are generated.
3. Login using your credentials.
4. Access the ATM Menu:

   * Balance Inquiry
   * Deposit Money
   * Withdraw Money
   * Fund Transfer
   * Change PIN
   * View Statements
   * Logout
5. All account information and transactions are stored locally in JSON files.
